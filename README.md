# Torchy
Built from scratch neural network library using only Numpy and PyTorch as reference to 

## Implemented
1. Layers
    - Linear (Dense) 
    - Convolutional
    - ReLU + BatchNorm
    - MaxPool (2D)
2. Optimizations
   - SGD;
   - MomentumSGD;
   - Adam;
   - RMRSProp;
   - Adagrad
3. Loss Functions
   - Cross-entropy Loss + Softmax
4. Schedulers
   - StepLR;
   - ReduceLROnPlateau
